# Simple Interest Calculator

## Description
A simple interest calculator that computes interest based on principal, rate, and time.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Variables
- **P** - Principal amount
- **R** - Rate of interest (per year)
- **T** - Time (in years)

## Example
P = 1000, R = 5, T = 3  
SI = (1000 × 5 × 3) / 100 = **150**
